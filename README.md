## ws1.2 สร้างตัวแปร5ตัวที่มีชนิด Type ต่างกัน แล้วใช้ typeof ในการเช็ค typeของตัวแปรแต่ละตัว และแสดงผลทาง console

## ws1.3 สร้างตัวแปร 3วิชา เพื่อนำมาหาค่ารวมและค่าเฉลี่ย ใช้ sum กับ average และใช้ comparison operatorในการเช็คค่า average ว่ามากกว่าหรือเท่ากับ80 มั้ย

## ws1.4 สร้างระบบตรวจสอบสิทธิ์การใช้งาน มี3ตัวแปร username password และ age โดยใช้ if else if เป็นหลักในการตรวจสอบเงื่อนไข เช่น username != "admin" || password != 1234 ให้แสดงว่า ชื่อผู้ใช้หรือรหัสผ่านไม่ถูกต้อง หรือif age< 18 ให้แสดงว่า อายุไม่ถึงเกณฑ์
